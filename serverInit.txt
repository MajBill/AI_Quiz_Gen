need "-pip install fastapi uvicorn openai"

fastapi - web framework for building api's, used wit uvicorn
uvicorn - Uvicorn is a web server. It handles network communication - receiving requests from 
    client applications such as users’ browsers and sending responses to them. It communicates
    with FastAPI using the Asynchronous Server Gateway Interface (ASGI), a standard API for Python web 
    servers that run asynchronous code.
openai - the official client software development kit (SDK) that allows developers to integrate OpenAI's 
    AI models directly into Python applications. Instead of writing low-level HTTP requests to interact 
    with OpenAI's cloud-based REST API, this module abstracts the complexity into simple, native Python 
    classes and functions.

-ContentType "application/json" `
